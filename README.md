# AL-RAWDA-Production-Sales-Operations-Performance
An interactive business intelligence dashboard developed to analyze production performance, sales insights, and operational efficiency. This project transforms raw operational data into meaningful insights using Excel and Power BI.AL RAWDA Production, Sales & Operations Performance Dashboard

The dashboard helps stakeholders monitor production output, revenue performance, operational costs, product demand, and logistics efficiency in a centralized analytical view.

 Tools & Technologies

Microsoft Excel – Data cleaning, transformation, and preparation

Power BI – Data modeling, DAX calculations, and interactive dashboard creation

GitHub – Project documentation and version control

 Project Dashboard Overview
 Home Page

This page provides navigation to all analytical sections including Production Analysis, Sales Overview, Operations Performance, and Insights & Recommendations.

Operations Performance Dashboard

Key operational metrics including water usage, operational cost, cost per bottle, machine downtime, transport dispatch, and damaged bottle analysis.

Production Analysis Dashboard

Insights into total production, monthly production trends, product type distribution, production line performance, and shift-wise production analysis.

Sales Overview Dashboard

Detailed sales insights including revenue trends, sales channel performance, regional revenue distribution, profit margin, and product revenue analysis.

Project Objective

The objective of this project is to demonstrate how data visualization and analytics can support operational monitoring, performance evaluation, and strategic decision-making in a manufacturing and sales environment.
